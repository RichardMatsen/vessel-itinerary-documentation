    # Project History

    The system was originally developed in 2008 as an add-on to the delivered MAPS Maritime Suite product for rostering vessel crew. MAPS is based on a mixture of technologies - VB6, C# and SmallTalk with a SQL Server 2000 database backend.

    The client requested that the interface be written in SQL Server DTS rather than a compiled language so that business logic changes could be adjusted by their own internal IT department.

    In 2011, the MAPS database component was upgraded to SQL Server 2005, which meant that the DTS component was perceived as a little outdated. It was decided to upgrade the internals of the DTS package to SSIS rather than run the DTS package using the Execute DTS control task.
