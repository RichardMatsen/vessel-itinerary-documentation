    # Vessel Itinerary Interface

    # Overview

    The LaraCIS Interface was written for CMA CGM French container transportation and shipping company to transfer Vessel Itinerary data from the Oracle LaraCIS Marketing database to the MAPS Vessel Rostering system.

    Key challenges were handling of inconsistencies in reference data (vessels and ports) and allowing manual changes in the target system that were not over-ridden on the next upload.

    The data is a sequence of port-call records with arrival and departure dates.
